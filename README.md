# 🚀 Filesterr SSR - How to Run

## 1. Install dependencies

npm install

---

## 2. Start development server

npm run dev

---

## 3. Open in browser

http://localhost:3001

---

## 4. Network access (optional)

If you want to access it from another device on the same network:

http://YOUR_LOCAL_IP:3001

(The IP will be shown in the terminal when the server starts.)

---

## ⚙️ Requirements

- Node.js 18+
- npm

---

## 🧠 Notes

- This project runs in SSR mode using Express + Vite
- Hot Module Replacement (HMR) is enabled
- WSL users: file watching uses polling automatically

---

## 🧾 Commands

npm install   → install dependencies  
npm run dev   → start development server
